# Benchmarks

## Overview

Opifex provides comprehensive benchmarking tools for evaluating scientific machine learning methods.

## PDE Benchmarks

### Heat Equation

Standard heat equation benchmarks with various boundary conditions and geometries.

### Navier-Stokes

Fluid dynamics benchmarks including:

- Lid-driven cavity
- Flow around cylinder
- Turbulent channel flow

## Neural Operator Benchmarks

Performance evaluation of different neural operator architectures on standard test problems.

## Quantum Chemistry Benchmarks

Validation against established quantum chemistry databases:

- QM9 dataset
- MD17 molecular dynamics
- Materials Project crystals

## Performance Metrics

- Accuracy vs reference solutions
- Computational efficiency
- Memory usage
- Scalability analysis
