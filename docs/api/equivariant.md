# Equivariant API Reference

Native E(3)-equivariant algebra and neural-network primitives used by the atomistic
and quantum-chemistry models (SchNet, PaiNN, NequIP, QHNet). Provides irreducible
representations, tensor products, gated nonlinearities, spherical harmonics, radial
bases, and graph utilities.
